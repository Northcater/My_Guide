While Loops¶

ถ้า For Loop มันจะวนซ้ำสมาชิกทุกตัวไปจนหมด¶

แต่ While Loop จะวนซ้ำไปตราบเท่าที่มันตรงกับ specific condition¶

ถ้าไม่ตรง condition ก็หยุดทันที ที่เหลือจะไม่ทำต่อ¶

In [32]:
number = 0

while number < 5:
    print(number)
    number = number + 1 # add 1 ให้ number ดังนั้นถ้าเกิน 5 มันก็จะหยุด

number = 0

while number < 5:
    print(number)
    if number == 3:
        continue # แปลว่าถ้า if true คือเลขเป็น 3 แล้วให้วนกลับไปที่ While อีกครั้ง
    number = number + 1
else:
    print('No longer < 5')
#ผลลัพธ์คือ พอมันยังไม่ใช่ if number == 3 ตัว number = number + 1 ก็จะยังทำต่อ
#แต่พอ if number == 3 ตัว continue จะทำงานแล้วให้วนกลับไปทำ while ก็คือ print 3 ออกมา
#ซึ่งแบบนี้แปลว่ามันก็จะวน 3 ไปเรื่อยๆไม่มีที่สิ้นสุด

# ทำแบบนี้ไป add เลขก่อน ก็จะไม่เกิดการวนซ้ำไม่มีที่สิ้นสุด

while number < 5:
    number = number + 1
    if number == 3:
        continue # พอเป็น 3 ก็วนกลับไปเลย ตัว print(number) ไม่ทำงาน
    print(number)
